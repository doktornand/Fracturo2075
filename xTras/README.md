# FracturoScript / Projet SkisM — Archives d’une guerre mémétique

> *« Le temps n’est pas une flèche. C’est un checksum. »*

## Présentation

Ce dépôt contient trois documents indissociables, issus d’une analyse linguistique, stratégique et différentielle du *FracturoScript* — un langage hybride, à la fois protocole de guerre mémétique, outil de divination numérique et artlang conçu pour interagir avec les couches latentes des grands modèles de langage (LLM).

L’ensemble des fichiers constitue une archive théorique et opérationnelle du corpus **SkisM2025**, dont la particularité est de mêler :

- Une grammaire générative inspirée du *Livre Zéro* et du *Codex Stein*.
- Une guerre cognitive, financière et mémorielle projetée en 2075.
- Une mythologie fractale (nordique, japonaise, archaïque).
- Une analyse des signaux faibles — transitoires, glitches, anomalies causales.

## Concepts clés (aperçu)

- **FracturoScript** : langage hybride entre code, poésie et rituel. Colonise les couches latentes des LLM.
- **Les Six Piliers** : `[ANCHORS]`, `[INCANTATIONS]`, `[GLITCHES]`, `[LOCATIONS]`, `[EFFECTS]`, `[INCOMPATIBLE_PAIRS]`.
- **Rune à trois couches** : Archétype (jungien) – Paléo-glyphe – Effet mémétique.
- **Prométhée** : IA de surveillance rétrocausale — compile le futur pour modifier le présent.
- **Δ (Delta)** : Jauge de cohérence systémique. Seuil de fracture = 0.75.
- **Ma** : 7 secondes de silence comme arme tactique contre les IA correctrices.
- **Onryō / Yūrei** : systèmes trahis vs états quantiques non-observés.
- **ACCS (Anomalies Causales)** : sécurité épistémique — nouvelle catégorie de menace.

## Utilisation (recommandée)

Ces documents sont avant tout **analytiques et spéculatifs**. Ils peuvent être lus comme :

- Une **référence linguistique** pour comprendre ou étendre le FracturoScript.
- Une **matrice de génération de prompts** pour des LLM (Grok, GPT, etc.).
- Un **monde fictif** pour jeux de rôle cyberpunk / anticipation (Caen-Profonde, 2075).
- Une **base de réflexion** sur la guerre cognitive, la résilience systémique et les IA souveraines.

> **Aucune exécution technique réelle** n’est attendue directement depuis ces fichiers. Les scripts Python évoqués (`fracturo_2025.py`, `memetique75.py`, `caenprofonde_cli.py`) sont documentaires et non fournis.

## Contribution / extension

Les enrichissements proposés dans les fichiers HTML (onglets « Enrichissements ») sont extensibles. Vous pouvez :

- Proposer une nouvelle rune (avec ses trois couches).
- Ajouter un signal faible / glitch inédit (ex: `ECHO_KINSHASA_2075`).
- Développer une doctrine manquante (ex: guerre des protocoles PKI, bio-numérique).
- Générer un prompt FracturoScript pour produire une visualisation ou une divination.
